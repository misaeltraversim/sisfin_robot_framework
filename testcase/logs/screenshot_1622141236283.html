 _______________________________________________________________________________
    The  purpose of this field is to select which screen you wish to access     
    next.   The selections include:                                             
                                                                                
        ? = Will give further explanations on the use of the field.             
        F = To view the Faculty/Staff Listings.                                 
        S = To view the Student Listings.                                       
        C = Will tell the viewer who to contact to correct information          
            in this system.                                                     
        B = To page backwards while viewing the listings.  This works only      
            for last name or department/office searches, not for first name     
            searches.                                                           
        M = Return to the Menu screen, this would be used if you wished to      
            change campuses.                                                    
        A = To view additional information, used only for the Faculty/Staff     
            Listings.  This will show the employees title and home phone number.
        Q = To end the session and sign off the system.
